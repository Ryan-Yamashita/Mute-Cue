# Changelog

All notable changes to Mute Cue are recorded here.

## Unreleased

- Compact, clearly selected Discord, BEACN, and Settings tabs.
- Always-visible BEACN fader-source list in the BEACN tab.
- Startup and optional start-in-system-tray settings.
- Faster authoritative state refreshes and guarded hotkey previews that follow BEACN Knob Mute assignments.
- Less alarming hardware synchronization feedback: normal confirmations no longer look like an error state.

## Beta release policy

Community beta releases are built from a tagged commit, signed, verified from the exact archive, and documented with their supported BEACN version range. See [RELEASE_CHECKLIST.md](RELEASE_CHECKLIST.md).
